TABLERO/CARTAS-NAV — CARTAS DE NAVEGACIÓN
═══════════════════════════════════════════════════════

Las cartas de navegación son las que juegan los jugadores en el cofre (fase_3).
Tienen 3 colores: azul (navegar), rojo (pirata), amarillo/dorado (cultista).

Archivos esperados:

  carta-azul.png            → carta de color azul (navegación adelante)
  carta-roja.png            → carta de color rojo (pirata / sabotaje)
  carta-amarilla.png        → carta de color amarillo (cultista / neutral)
  carta-reverso.png         → reverso genérico (cuando la carta está boca abajo)

Todos PNG con transparencia. Tamaño sugerido: 140×200 px (proporción carta estándar).
