Archivos esperados en esta carpeta:

  fondo.png
    Imagen de fondo de la escena completa (1920×1080).
    Ejemplo: cubierta de barco, océano tormentoso, sala de mando.

  marco-tablero.png
    Marco/ornamento que rodea el tablero hexagonal central.
    Se coloca encima del SVG del tablero. Fondo transparente (PNG).
    Tamaño sugerido: cuadrado, ~800×800 px o más grande.

  tablero-mapa.png  (opcional)
    Si prefieres usar una imagen PNG para el tablero en lugar del SVG generado.
    Se usa como fondo dentro del marco. Fondo transparente (PNG).

Subcarpetas:
  ui/            → botones, paneles, decoraciones de la interfaz
  fichas/        → tokens de jugadores, barco, marcadores
  cartas-nav/    → imágenes de las cartas de navegación (azul, rojo, amarillo)
  cartas-ritual/ → imágenes de las cartas rituales del Culto
